# Hawk Compiler Toolchain

## Building with Stack

Install stack and then initalize the project with

```bash
$ stack setup
$ stack build
```

Once you have a successful build, you can run the following commands.

``` sh
# Build the project.
stack build

# Run the test suite.
stack test

# Run the benchmarks.
stack bench

# Generate documentation.
stack haddock
```
